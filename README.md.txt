A desktop banking system built with VB.NET Windows Forms and SQL Server Express.


 Features
	•	Admin Login
	•	Customer Management
	•	Account Creation
	•	Deposit & Withdraw
	•	Transaction History
	•	Admin Dashboard


 Setup Instructions
	1.	Install SQL Server Express
	2.	Open Database/BankDBQuery.sql
	3.	Run it in SQL Server Management Studio
	4.	Open the project in Visual Studio
	5.	Update connection string if needed:
        6.	Run the application
